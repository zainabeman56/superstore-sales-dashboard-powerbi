# SuperStore Sales Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive Sales Dashboard built using Microsoft Power BI to analyze SuperStore sales performance. The dashboard provides valuable business insights into sales, profit, shipping performance, customer segments, payment modes, and regional performance.

The objective of this project is to transform raw sales data into meaningful visual insights that support data-driven decision making.

---

## 📂 Dataset

**Dataset:** SuperStore Sales Dataset

The dataset contains information about:

- Sales
- Profit
- Orders
- Customers
- Shipping
- Products
- Regions
- Categories
- Payment Modes

---

## 🛠 Tools Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling

---

## 📈 Dashboard Features

### KPI Cards
- Total Sales
- Total Profit
- Total Orders
- Average Ship Days

### Interactive Filters
- Region Slicer

### Visualizations
- Sales by Segment
- Sales by Payment Mode
- Monthly Sales Comparison (Year-over-Year)
- Monthly Profit Comparison (Year-over-Year)
- Sales by Ship Category
- Sales by Category
- Sales by Sub-Category
- Sales and Profit by State (Map)

---

## 📊 Key Business Insights

- Standard Class shipping contributes the highest sales.
- Technology is one of the highest revenue-generating product categories.
- Phones are the top-selling sub-category.
- Sales show noticeable growth during the final months of the year.
- Regional filters allow users to compare performance across different regions.
